CSS variables are declared within the element style block.

Syntax : --[var_name]: [var_value];

The scope of variable is within the block and children elements. However, a variable declared within the child element cannot be accessed by the parent.

To create a global var, we define the variable inside the html sytle block.

CSS variables comes in handy while editing the laayouts, since we dont have to change the values again and again.
